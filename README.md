# Excel Pivot Table Analysis – Sales Performance Report

## Project Overview
This project demonstrates data analysis using Excel Pivot Tables to summarize,
analyze, and extract meaningful business insights from a sales dataset.
It is designed to showcase practical Excel skills required for data analyst
and business analyst internship roles.

## Objective
- Perform sales analysis using Pivot Tables
- Identify top-performing categories, regions, and segments
- Calculate profit margins and analyze profitability
- Build interactive dashboards using slicers
- Present insights in a clear and business-friendly format

## Tools Used
- Microsoft Excel / Google Sheets
- Pivot Tables and Pivot Charts
- Slicers and Conditional Formatting
- PDF and Document Export

## Analysis Performed
### Data Cleaning
- Removed missing and duplicate values
- Standardized category, region, and segment names
- Fixed incorrect date and numeric formats

### Sales Analysis
- Total Sales by Category
- Sales by Region and Segment
- Identification of business-driving segments

### Profitability Analysis
- Created calculated column: Profit Margin = Profit / Sales
- Analyzed average profit margin by category and region

### Visualization and Dashboard
- Pivot charts for category-wise sales comparison
- Slicers for interactive filtering
- Conditional formatting to highlight performance

## Key Business Insights
- West region contributes the highest percentage of total sales.
- Technology category is the primary revenue driver.
- South region is consistently underperforming.
- Consumer segment shows stable sales but lower profitability.
- Profit margins vary significantly across regions.
